# 2444

[코딩테스트 연습 - 2444][1] 로 이동

## 문제

예제를 보고 규칙을 유추한 뒤에 별을 찍어 보세요.

## 입력

첫째 줄에 N(1 ≤ N ≤ 100)이 주어진다.

## 출력

첫째 줄부터 2×N-1번째 줄까지 차례대로 별을 출력한다.

## 예제 입력

```
5

```

## 예제 출력

```
    *
   ***
  *****
 *******
*********
 *******
  *****
   ***
    *

```

## 풀이

정방향으로 for를 해준 후, 역방향으로도 해준다.

```python
N = int(input())

for n in range(N):
    print(" " * (N - n - 1), "*" * (1 + (n * 2)), sep="")
for n in range(N - 2, -1, -1):
    print(" " * (N - n - 1), "*" * (1 + (n * 2)), sep="")

```
