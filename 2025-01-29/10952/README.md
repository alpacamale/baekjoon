# 10952

[코딩테스트 연습 - 10952][1] 로 이동

## 문제

두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

## 입력

입력은 여러 개의 테스트 케이스로 이루어져 있다.

## 출력

각 테스트 케이스마다 A+B를 출력한다.

## 예제 입력

```
1 1
2 3
3 4
9 8
5 2
0 0
```

## 예제 출력

```
2
5
7
17
7
```

## 풀이

A과 B가 모두 0일때까지 반복해주면 된다.

```python
while True:
    A, B = [int(num) for num in input().split()]
    if A == B == 0:
        break
    print(A + B)

```
