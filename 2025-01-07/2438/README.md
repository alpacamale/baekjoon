# 2438

[코딩테스트 연습 - 2438][1] 로 이동

## 문제

첫째 줄에는 별 1개, 둘째 줄에는 별 2개, N번째 줄에는 별 N개를 찍는 문제

## 입력

첫째 줄에 N(1 ≤ N ≤ 100)이 주어진다.

## 출력

첫째 줄부터 N번째 줄까지 차례대로 별을 출력한다.

## 예제 입력

```
5
```

## 예제 출력

```
*
**
***
****
*****
```

## 풀이

반복문을 사용하여 쉽게 풀 수 있다.

```python
N = int(input())
for number in range(1, N + 1):
    print("*" * number)
```
